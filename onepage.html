GUIDELINES

How to Read
the guidelines cards

Each guideline card has been created
according to the following criteria:


rule of action, containing about 3-4 words
and starting with a verb

one-sentence description expresses in detail the action to be performed by the conversational agent

symbol that refers to the user-journey phase in which the guideline applies

one-sentence statement of the user that proposes an example of correct application of the guideline

one-sentence statement of the user that proposes an example of incorrect application of the guideline

SCENARIOS

How to Read
Scenarios Cards

Each card is associated to a situational conversation defined Scenario: example of correct application of the guideline in specific context selected to correctly represent the application of the guideline. The Scenario is composed by:


Website

The website model in which the action occurs

Pages

The website’s page considered

Action

The situation happening in the scenario

Website considered

Types of Websites selected
for this study

mostly journalistic websites that function as news containers, constantly updated to provide news in real-time

Websites that present different outcomes
based on the input provided by the user

mostly used to present universities or agencies,
that provide information about a specific topic
