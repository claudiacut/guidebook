About this project

What are these guidelines?

The cards give instructions on how to realize CAs for translating the Web experience from a visual to a conversational navigation

Where did the research start?

We conducted research with blind and visually impaired (BVI) users to identify which parts of screen reader web browsing are frustrating, and in general what can be improved

How where the guidelines found?

Starting from the user research, we were able to identify a total of 6 challenges that user experience while using screen reader and CAs for Web Navigation
